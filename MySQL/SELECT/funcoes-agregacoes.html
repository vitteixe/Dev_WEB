Funções de agregações

São definidadas no bloco SELECT das querys

Funções de agregações, tem como objetivo permitir uma operação artimetica sobre os valores de uma determinada coluna, em todos os registros de uma tabela.




MIN

MIN retorna o MENOR valor de determinada coluna





MAX

MAX retorna o MAIOR valor de determinada coluna





AVG

AVG retorna a MÉDIA de valores de determinada coluna
