Criando e excluindo BD

Criar: CREATE DATABASE nome_do_banco

Excluir: DROP DATABASE nome_do_banco


Tipos de dados

Campos de TEXTO

Text: (Tamanho variável que armazena uma grande quantidade de caracteres)
Varchar: (Tamanho variável que armazena de 0 até 255 caracteres)
Char: (Tamanho FIXO que que armazena de 0 até 255 caracteres)

Campos NUMÉRICOS

Int: (Valores numéricos inteiros)
float: (Valores Numéricos Fracionados)

Campos de DATA e HORA

Date: (data no formato YYYY/mm/dd)
Time: (hora)
Datetime: (combinação de date e time em um mesmo campo)



Tabelas

Para criar uma tabela, basta utilizar o comando CREATE TABLE nome_da_tabela ( ); para escluir basta utilizar DROP

Dentro dos parenteses, é acrescentado o nome das colunas nome_coluna - tipo - null

Exemplo:

CREATE TABLE tb_cursos(

id_cursos int not null,

image_curso varchar(100) not null,

);
